# FastAPI do Zero

## Configurando o banco de dados e gerenciando migrações com Alembic

### SQLalchemy

* É um ORM (Mapeamento Objeto-Relacional), que é um técnica de programação que vincula objetos a registros de banco de dados
* É recomendado utilizar um ORM poi ele permite:
    * Abstração de banco de dados: permite muda um tipo de banco de para sem precisar alterar o código
    * Segurança: previne injeções SQL
    * Eficoência no desenvolvimento
